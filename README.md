## 闪电记账
[项目预览地址](http://zhq691150.gitee.io/tset/#/money)

#### 介绍
闪电记账是一款基于 TypeScript、React Hooks、React Router 的移动端单页面应用。特点在于一键极速记账，为满足我的日常记账需求而设计。

#### 技术栈
* TypeScript
* React Hooks
* React-Router
* Webpack
* Styled-Component
